# tanchishe
# 贪吃蛇游戏 🐍
经典贪吃蛇游戏的现代化实现，包含用户系统和游戏日志功能。采用C++开发，适用于Windows控制台环境。

## 功能特性 ✨

- **用户管理系统** 👤
  - 注册/登录功能
  - 用户数据持久化存储
  - 自动登录机制

- **游戏核心玩法** 🎮
  - 支持方向键控制
  - 动态难度调节（F1加速/F2减速）
  - 实时分数显示
  - 碰撞检测（墙体/自噬）

- **数据统计功能** 📊
  - 游戏时长记录
  - 历史战绩查询（F5）
  - 数据文件存储（users.dat / gamelogs.dat）

- **界面优化** 🖥️
  - 控制台图形化界面
  - 右侧信息面板
  - 游戏暂停/继续功能

## 技术栈 🛠️

- **开发语言**: C++
- **系统依赖**: Windows API
- **数据结构**: 链表（蛇体/用户/日志）
- **持久化存储**: 二进制文件操作
- **时间处理**: `<time.h>`库
